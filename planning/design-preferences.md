# Free Typing Camp Design Preferences

Last updated: May 2, 2026

The preferred visual direction for Free Typing Camp is warm, welcoming, and education-focused, with a light camp/nature theme. The site should feel like a thoughtfully designed learning product rather than a generic AI-generated dashboard.

Key preferences:
- Use flat color as the default. Gradients should be rare, subtle, and purposeful.
- Lean toward warm cream/paper backgrounds, deep navy or camp-night header bands, campfire orange accents, muted pine/leaf greens, and occasional soft violet.
- Favor roomy editorial sections, clean cards, and clear information hierarchy similar to edclub/TypingClub pages.
- Preserve playful illustration and mascot energy, but keep it as an accent rather than clutter.
- Reference style touchpoints: IMG_8487, IMG_8485, `www.edclub.com_schools (1)`, and selected parts of the earlier mobile UI references.
- The brand should eventually support camping-themed icons, keyboard layouts, keycaps, and unlockable visual themes.
- The typing page should stay focused and immediate: minimal header, compact settings, large readable typing area, and the custom keyboard visible by default with an option to hide it.
- Responsive keyboard behavior matters: large screens should default to a full keyboard layout, small screens should default to a compact mobile key layout, and users should be able to force either view or hide the keyboard.
- Lesson menus should preserve labeled levels and mastery stars so users can scan completion and performance before opening a lesson.
- Games should feel like focused typing drills, reusing the same input/highlight/result systems as tests and lessons. The old calculator game is a good pattern for lives, score, difficulty, and a clear "Start Typing!" ready state.

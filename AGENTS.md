# Free Typing Camp Agent Notes

## Design Rules

- Follow the existing Free Typing Camp visual system unless the user explicitly asks to deviate.
- Page styling should not use visible borders. Prefer whitespace, alignment, surface contrast, soft fills, and soft shadow. Do not add border utilities, border-color states, or border divider lines unless the user explicitly asks for a border.
- Keep the palette warm and cohesive: cream/off-white backgrounds, deep navy structure, orange/coral primary accents, and muted sage/peach/tan support colors.
- Avoid old bright-blue UI treatments and avoid nested boxes-within-boxes.
- New interactive states should use the shared design language: orange for primary/next action, sage for correct/home-row/supportive states, peach/coral for warning or error.
